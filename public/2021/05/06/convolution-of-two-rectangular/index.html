дёӨдёӘзӣёеҗҢзҡ„зҹ©еҪўдҝЎеҸ·еҒҡеҚ·з§Ҝ / Convolution of two identical rectangular signals

иғҢжҷҜ

жҷҡдёҠеҒҡдҝЎеҸ·дёҺзі»з»ҹд№ йўҳзҡ„ж—¶еҖҷпјҢйҒҮеҲ°дәҶдёҖдёӘеҚ·з§Ҝи®Ўз®—зҡ„й—®йўҳпјҢеҸ‘зҺ°еңЁжҲ‘еҜ№е®ғзҡ„зҗҶи§ЈеҮәзҺ°дәҶдёҖзӮ№еҘҮеҰҷзҡ„еҒҸе·®пјҢйҖҡиҝҮзҪ‘з»ңиҙ№дәҶеҘҪдәӣеҠҹеӨ«жүҚжўізҗҶйҖҡгҖӮзҗҶи§Јд»ҘеҗҺпјҢе°ұжғіеҶҷдёҖзҜҮеҚҡж–Үи®°еҪ•дёӢиҝҷдёҖй—®йўҳпјҢеёҢжңӣиғҪз»ҷеӨ§е®¶дёҖзӮ№её®еҠ©пјҢйЎәдҫҝйҳІжӯўжҲ‘д»ҘеҗҺеҸҲеҝҳи®°дәҶгҖӮ

й—®йўҳ

йўҳзӣ®иҰҒжұӮзҡ„и®Ўз®—иҝҮзЁӢдёӯпјҢжңүи®Ўз®—еҰӮдёӢдёӨдёӘзӣёеҗҢзҹ©еҪўи„үеҶІдҝЎеҸ·зҡ„еҚ·з§Ҝпјҡ

еӣҫдёҖпјҢеҸ–иҮӘеҸӮиҖғй“ҫжҺҘ[1]

иҷҪз„¶жҲ‘зҹҘйҒ“е®ғзҡ„з»“жһңпјҢжҳҜдёҖдёӘеҰӮдёӢеӣҫжүҖзӨәзҡ„дёүи§’еҪўи„үеҶІдҝЎеҸ·вҖ”вҖ”иҜҫдёҠдёҺд№ҰдёҠйғҪжҳҜиҝҷд№ҲиҜҙзҡ„пјҡ

еӣҫдәҢпјҢеҗҢж ·еҸ–иҮӘеҸӮиҖғй“ҫжҺҘ[1]

еҸҜжҳҜз”ұдәҺжІЎжңүз»ҷиҝҮзЁӢпјҢжүҖд»ҘжҲ‘жғідёҚеҲ°жӯЈзЎ®зҡ„и®Ўз®—ж–№ејҸжҳҜд»Җд№ҲпјҢд№ҹе°ұж— д»Һж”№й”ҷгҖӮ

жҖқи·Ҝ

з”ұдёҠиҝ°еҸҜзҹҘпјҢжңүдёӨдёӘзӣёеҗҢзҡ„зҹ©еҪўи„үеҶІдҝЎеҸ·еҸҜиЎЁзӨәдёәпјҡ

$$ \tag{1.1} f(x) = \begin{cases} A, &\text{if } ~ {-T_0 \lt t \lt T_0} \\ 0, &\text{if } ~ {|t| ~ \gt T_0} \end{cases} ~ = ~ g(t) $$

иҖҢеҪ“дҝЎеҸ·зӣёдә’еҚ·з§Ҝж—¶пјҢиҝһз»ӯж—¶й—ҙдёӢпјҢз”ұеҚ·з§Ҝз§ҜеҲҶзҡ„е…¬ејҸпјҡ

$$ \tag{1.2} y(t) = \int_{-\infty}^{+\infty}{x(\tau)}{h(t-\tau)}{\rm d}{\tau} ~ = ~ x(t){\ast}h(t) $$

дёӨиҖ…зӣёеҚ·з§Ҝж—¶пјҢеңЁеӣҫи§Јж—¶еҸҜе°Ҷзӣёеә”жІҝ$y$иҪҙзҝ»иҪ¬пјҢз„¶еҗҺе·Ұз§»дёҖе®ҡзҡ„и·қзҰ»зӣҙеҲ°дёӨдёӘдҝЎеҸ·дёҚеҶҚйҮҚеҸ пјҢеҶҚйҖҗжёҗеҸіз§»пјҢйҮҚеҸ еҗҺйҖҗзӮ№и®Ўз®—еҚ·з§ҜеҖјгҖӮпјҲиҜҰз»Ҷзҡ„и®Ўз®—иҝҮзЁӢй»ҳи®ӨиҜ»иҖ…е·Із»ҸдәҶи§ЈпјҢеҰӮдёҚдәҶи§ЈеҸҜеҸӮи§ҒеҘҘжң¬жө·е§Ҷзҡ„гҖҠдҝЎеҸ·дёҺзі»з»ҹгҖӢз¬¬дәҢз« пјү

иҖҢеңЁжӯӨеӨ„пјҢз”ұдәҺдёӨдёӘдҝЎеҸ·зӣёеҗҢпјҢжүҖд»Ҙе…¶еҚ·з§ҜеҸҜеҶҷдёә$y(t) = x(t){\ast}x(t)$пјҢд»»йҖүе…¶дёҖдҪңдҝЎеҸ·зҡ„е“Қеә”$x_2(t)$пјҢж•…е…¶жІҝ$y$иҪҙеҸҚиҪ¬еҗҺдҝқжҢҒдёҚеҸҳпјҢеңЁе·Ұз§»жІЎжңүйҮҚеӨҚеҢәеҹҹеҗҺеҶҚж¬ЎеҸіз§»зҡ„иҝҮзЁӢдёӯпјҢдёҺдҝЎеҸ·$x_1(t)$жңҖж—©зӣёжҺҘи§Ұзҡ„ең°ж–№еә”иҜҘжҳҜ$t = -T_0$зҡ„ж—¶еҖҷпјҢиҖҢжӯЈзЎ®зҡ„з»“жһңеә”иҜҘжҳҜ$t = -2T_0$гҖӮ

ObviouslyпјҢиӮҜе®ҡжңүең°ж–№еҮәдәҶй—®йўҳгҖӮ

зә й”ҷ

еӣһйЎҫд№ӢеүҚеҒҡзҡ„дҫӢйўҳдёӯпјҢе…¶е“Қеә”дёҖиҲ¬йғҪжҳҜд»Һйӣ¶зӮ№ејҖе§Ӣзҡ„пјҢд»ҺжқҘжІЎжңүд»Һ$t \lt 0$зҡ„жғ…еҶөпјӣдё”еҪ“дёӨдёӘзҹ©еҪўи„үеҶІдҝЎеҸ·йғҪжҳҜд»Һ$t = 0$ејҖе§Ӣж—¶пјҢжҲ‘еҸҜд»Ҙи®Ўз®—еҫ—еҲ°жӯЈзЎ®зҡ„иҫ“еҮәпјҢдёәдёҖдёӘд»Һ$t = 0$еҲ°$t = 4T_0$зұ»дјјеӣҫдәҢзҡ„дёүи§’еҪўпјҢжүҖд»ҘиӮҜе®ҡжҳҜеӣ дёәжҲ‘еҜ№е“Қеә”еӨ„зҗҶдёҚеҪ“иҖҢдә§з”ҹзҡ„й”ҷиҜҜгҖӮ

дёәжӯӨпјҢжҲ‘зҝ»д№ҰжғізңӢзңӢжңүжІЎжңүзӣёдјјзҡ„дҫӢйўҳпјҢж— жһңпјҢйҡҸеҗҺдёҠдёҮиғҪзҡ„GoogleиҜ•иҜ•иғҪдёҚиғҪжҗңеҲ°зӣёдјјжғ…еҶөзҡ„и§ЈжһҗгҖӮдёӯж–ҮжҗңзҙўеҮәжқҘе…ЁйғҪжҳҜCSDNжҲ–иҖ…еҚҡе®ўеӣӯзҡ„йӣ·еҗҢж–Үз« пјҢдёҖзӮ№её®еҠ©йғҪжІЎжңүвҖҰвҖҰжҚўдәҶиӢұж–Үе…ій”®иҜҚеҗҺпјҢжүҫеҲ°дәҶеҗҺйқўеҸӮиҖғж ҸйҮҢзҡ„дёӨдёӘзҪ‘йЎөпјҢзңӢеҗҺжҲ‘зҹҘйҒ“жҳҜе“ӘйҮҢеҮәй”ҷдәҶпјҡ

йҰ–е…Ҳз”ұйўҳж„ҸпјҢдҪңеҮәж–№зЁӢпјҡ

$$ \tag{2.1} y(t) = \int_{-\infty}^{+\infty}{x_1(\tau)}{x_2(t-\tau)}{\rm d}{\tau} $$

з”ұеӣҫи§Јжі•еҸҜдҪңеҮәдёӢеӣҫпјҡ

жіЁж„Ҹпјҡе“Қеә”$x_2(t)$дёҺиҫ“е…ҘдҝЎеҸ·$x_1(t)$зӣёд№ҳз§ҜеҲҶзҡ„з»“жһңеңЁ$t = t$зҡ„ж—¶еҖҷпјҢиҖҢдёҚжҳҜ$t = t + 1$еӨ„пјҢиҝҷжҳҜе…ій”®зҡ„дёҖжӯҘгҖӮд№ӢеүҚжҲ‘зҡ„й”ҷиҜҜе°ұжҳҜе°Ҷе…¶з§ҜеҲҶд№Ӣе’ҢиҜҜж”ҫеңЁдәҶ$t = t + 1$зҡ„ең°ж–№пјҢеҜјиҮҙиҫ“еҮәдҝЎеҸ·ејҖе§ӢдәҺ$t + 1$иҖҢдёҚжҳҜ$t$гҖӮ

д№ӢеҗҺдҫҝдёҖеёҶйЈҺйЎәдәҶгҖӮз©¶е…¶й”ҷиҜҜеҺҹеӣ пјҡжҲ‘д№ӢжүҖд»ҘдјҡйӮЈж ·и®Ўз®—пјҢйҰ–е…ҲжҳҜе…Ҳе…Ҙдёәдё»пјҢи®Өдёәд№ӢеүҚйғҪжҳҜд»Һе“Қеә”йҮҚеҸ зҡ„ең°ж–№ејҖе§ӢпјҢжғіеҝ…иҝҷж¬Ўд№ҹеә”иҜҘжҳҜиҝҷж ·пјӣе…¶ж¬ЎжҳҜеҜ№жҰӮеҝөзҡ„зҗҶи§Јиҫғдёәи–„ејұпјҡ$h(t - \tau)$д»ЈиЎЁдәҶеңЁ$t = \tau$еӨ„дёҖдёӘж—¶з§»еҗҺзҡ„и„үеҶІдҝЎеҸ·$\delta(0)$зҡ„е“Қеә”пјҢеӣ жӯӨ$x_2(t-\tau)$д№ҹеә”иҜҘд»ЈиЎЁеңЁ$t = \tau$еӨ„дёҖдёӘж—¶з§»еҗҺзҡ„и„үеҶІдҝЎеҸ·$\delta(0)$зҡ„е“Қеә”пјҢе…¶е“Қеә”еңЁ$t$еӨ„пјҢиҖҢйқһжңҖеҸіз«Ҝ$t+1$еӨ„гҖӮ

еӣҫеғҸеҫ—еҲ°дәҶпјҢиҮідәҺи§ЈжһҗејҸзҡ„з»“жһңпјҢжҲ‘д№ҹдёҖе№¶йҷ„еңЁиҝҷйҮҢеҗ§пјҡ

$$ \begin{split} x_1(t)*x_2(t) &= {\int_{-\infty}^{+\infty}{x_1(\tau)}{x_2(t-\tau)}{\rm d}{\tau}} \\ &=\begin{cases} {\int_{-T_0}^{t+T_0}1{\cdot}1{\rm d}{\tau}}, &\text{if } ~ {-2T_0 \lt t \lt 0} \\ {\int_{t-T_0}^{T_0}1{\cdot}1{\rm d}{\tau}}, &\text{if } ~ {0 \lt t \lt 2T_0} \end{cases} \\ &=\begin{cases} {t+2T_0}, &\text{if } ~ {-2T_0 \lt t \lt 0} \\ {2T_0-t}, &\text{if } ~ {0 \lt t \lt 2T_0} \\ {0}, , &\text{if } ~ {|t| \gt 2T_0} \end{cases} \end{split}\tag{2.2} $$

еӨ§жҰӮе°ұжҳҜиҝҷж ·дәҶпјҢеҰӮжһңж„ҹи§үзҗҶи§Јиө·жқҘжңүдәӣеӣ°йҡҫзҡ„иҜқпјҢжҺЁиҚҗеӨ§е®¶зңӢзңӢз¬¬дәҢдёӘй“ҫжҺҘдёӯзҡ„YouTubeи§Ҷйў‘пјҢжғ…еҶөдёҚе®Ңе…ЁзӣёеҗҢпјҢдёҚиҝҮAdamе°Ҹе“Ҙи®Іеҫ—и¶…жЈ’зҡ„пјҒ

йҷ„еҪ•

иҝҷжҳҜжҲ‘з¬¬дёҖж¬ЎеңЁеҚҡж–ҮйҮҢеөҢеҘ—е…¬ејҸпјҢиҝҷйҮҢйҷ„дёҠдёҠж–ҮжүҖеҮәзҺ°е…¬ејҸзҡ„д»Јз ҒгҖӮиҝҷеӨӘйҡҫжү“дәҶпјҢиҰҒдёҚжҳҜдёәдәҶеҘҪзңӢжҲ‘е°ұжүӢеҶҷдәҶвҖҰвҖҰйЎәжүӢйҷ„еңЁиҝҷйҮҢпјҢд»ҘеҗҺжҲ‘жңүйңҖиҰҒпјҲеҝҳи®°дәҶпјүзҝ»еӣһжқҘзңӢд№ҹж–№дҫҝгҖӮ

# ејҸ1.1
\tag{1.1}
f(x) = \begin{cases}
   A, &\text{if } ~ {-T_0 \lt t \lt T_0} \\
   0, &\text{if } ~ {|t| ~ \gt T_0}
\end{cases}
~ = ~ g(t)
# ејҸ1.2
\tag{1.2}

# ејҸ2.1
\tag{2.1}
y(t) = \int_{-\infty}^{+\infty}{x_1(\tau)}{x_2(t-\tau)}{\rm d}{\tau}
# ејҸ2.2
\begin{split}
x_1(t)*x_2(t) &= {\int_{-\infty}^{+\infty}{x_1(\tau)}{x_2(t-\tau)}{\rm d}{\tau}} \\
  &=\begin{cases} 
  	{\int_{-T_0}^{t+T_0}1{\cdot}1{\rm d}{\tau}}, &\text{if } ~ {-2T_0 \lt t \lt 0} \\
  	{\int_{t-T_0}^{T_0}1{\cdot}1{\rm d}{\tau}}, &\text{if } ~ {0 \lt t \lt 2T_0}
  	\end{cases} \\
  &=\begin{cases}
  		{t+2T_0}, &\text{if } ~ {-2T_0 \lt t \lt 0} \\
  		{2T_0-t}, &\text{if } ~ {0 \lt t \lt 2T_0} \\
  		{0}, , &\text{if } ~ {|t| \gt 2T_0}
  	\end{cases}
\end{split}\tag{2.2}
